# 海南大学毕业论文（设计）格式要求

> 依据《2022毕业论文模板草稿_计算机2022》及《附件12毕业论文设计排版打印格式》整理

---

# 第一部分：格式要求（纯学校规定）

## 1. 页面设置

| 项目 | 设置值 |
|------|--------|
| 纸张 | A4 |
| 上 (T) | 2.54 cm |
| 下 (B) | 2.54 cm |
| 左 (L) | 3.17 cm |
| 右 (R) | 3.17 cm |
| 装订线 | 1.0 cm |
| 装订线位置 | 左 |
| 打印方向 | 纵向 |
| 页眉 | 除封面和封底外，每页均应带有页眉（使用论文题目） |
| 页码位置 | 每页底部中间 |
| 摘要与目录页码 | I、II、III...（大写罗马数字） |
| 正文页码 | 从 1 开始连续编号 |

---

## 2. 论文结构

论文由以下部分组成，各部分之间使用"**分节符（下一页）**"。

| 序号 | 部分 | 说明 |
|------|------|------|
| 1 | 封面 | 计算机打印 |
| 2 | 中文摘要 | 单独一页，不少于300字 |
| 3 | 英文摘要（Abstract） | 单独一页，与中文对应 |
| 4 | 目录 | 单独一页，使用Word自动插入功能 |
| 5 | 正文 | 页码从1开始 |
| 6 | 致谢 | |
| 7 | 参考文献 | 五号字 |
| 8 | 附件 | 若无可省略 |

---

## 3. 封面格式

| 内容 | 字体 | 字号 | 说明 |
|------|------|------|------|
| 海南大学 | 宋体 | 一号 | |
| 毕业论文（设计） | 黑体 | 一号 | |
| 题目 | 宋体 | 三号 | 若太长按回车换行 |
| 学号、姓名、年级、学院、系别、专业、指导教师、完成日期 | 宋体 | 三号 | |

---

## 4. 摘要格式

### 4.1 中文摘要（论文第二页）

| 内容 | 字体 | 字号 | 格式 | 字数要求 |
|------|------|------|------|----------|
| 摘 要 | 宋体 | 三号 | 加粗、居中 | |
| 摘要正文 | 宋体 | 小四 | 1.5倍行距 | **不少于300字** |
| 关键词： | 宋体 | 小四 | 顶格、加粗 | 3~5个 |
| 关键词内容 | 宋体 | 小四 | 词间用分号隔开 | |

### 4.2 英文摘要（论文第三页）

| 内容 | 字体 | 字号 | 格式 |
|------|------|------|------|
| Abstract | Times New Roman | 三号 | 加粗、居中 |
| Abstract正文 | Times New Roman | 小四 | 1.5倍行距 |
| Keywords: | Times New Roman | 小四 | 顶格、加粗 |
| Keywords内容 | Times New Roman | 小四 | 词间用分号隔开 |

---

## 5. 目录格式

| 内容 | 字体 | 字号 | 格式 |
|------|------|------|------|
| 目 录 | 宋体 | 三号 | 加粗、居中 |
| 目录内容 | 宋体 | 小四 | 行距1.5倍 |

**注意**：
- 一级标题：三号、宋体、顶格、加粗
- 二级标题：四号、宋体、顶格、加粗
- 三级标题：小四、宋体、顶格、加粗
- 内容调整后需右击目录选择"更新域"

---

## 6. 正文标题格式

| 级别 | 格式示例 | 字体 | 字号 | 格式 | 行距 |
|------|---------|------|------|------|------|
| 一级标题 | 第1章 绪论 | 宋体 | 三号 | 顶格、加粗 | **保留自带行距，非1.5倍** |
| 二级标题 | 1.1 研究背景 | 宋体 | 四号 | 顶格、加粗 | **保留自带行距，非1.5倍** |
| 三级标题 | 1.1.1 具体问题 | 宋体 | 小四 | 顶格、加粗 | **保留自带行距，非1.5倍** |
| 四级标题 | (1) 具体问题 | 宋体 | 小四 | 顶格、加粗 | **保留自带行距，非1.5倍** |
| 五级标题 | 具体问题 | 宋体 | 小四 | 首行缩进2字符、加粗 | **段前段后0，行距1.5倍** |

**重要提醒**：正文中不能有红色字体（模板中为强调用，实际应为黑色）。

## 10. 参考文献格式

### 10.1 标题与正文字体

| 内容 | 字体 | 字号 | 格式 |
|------|------|------|------|
| 参考文献 | 宋体 | 三号 | 居中、加粗 |
| 参考文献内容 | 宋体 | 五号 | 英文用五号 Times New Roman |

### 10.2 三种引用格式

| 类型 | 格式 |
|------|------|
| 专著、论文集、报告、学位论文 | `[序号] 作者(列前3名).文献名.出版社所在地：出版社，出版年.起始页-终止页.` |
| 期刊文章 | `[序号] 作者(列前3名).论文名.刊名，出版年，卷(期)：起始页-终止页.` |
| 电子文献 | `[序号] 作者(列前3名).电子文献名.电子文献的出处或可获得地址，发表或更新日期.` |

### 10.3 正文引用

- 格式：`[序号]`，设置为**上标**
- 序号**不包含章节号**，从正文开始排序
- 示例：该算法已被广泛应用^[1]^。

| 附件 | 三号、宋体、居中、加粗 | 文字：小四宋体；代码：五号（中文宋体、英文Times New Roman） |

# 第二部分：写作指南（Markdown相关）

## A. Markdown 标题层级对应（配合 pandoc）

```markdown
# 第1章 绪论                    → 一级标题（三号）
## 1.1 研究背景                 → 二级标题（四号）
### 1.1.1 具体问题             → 三级标题（小四）
#### (1) 具体问题              → 四级标题（小四）
##### 具体问题                  → 五级标题（小四，需手动调整首行缩进和行距）
```

**注意**：pandoc 转换后，一级到四级标题需在 Word 中调整为"保留自带行距"（非1.5倍）。

---

## B. 图表公式在 Markdown 中的写法

### B.1 表格

```markdown
如表1所示，系统性能...

| 指标 | 数值 |
|------|------|
| 延迟 | 50ms |

表1 系统性能指标
```

- 表题在表格**上方**
- 表序**不包含章节号**

### B.2 图片

```markdown
如图1所示，架构设计...

![图1](architecture.png)

图1 系统架构图
```

- 图题在图片**下方**
- 图序**不包含章节号**

### B.3 公式

```markdown
质能方程如公式1所示：

$$E = mc^2$$                              （公式1）
```

---

## C. 参考文献在 Markdown 中的写法

```markdown
该算法已被广泛应用^[1]^。

## 参考文献

[1] 张三，李四.计算机网络安全.北京：清华大学出版社，2020.15-30.
[2] 王五.区块链技术研究.计算机学报，2019，42(5)：100-115.
```

**注意**：pandoc 转换后需手动将 `[1]` 设置为上标格式。

---

## D. Pandoc 转换命令

```bash
# 基础转换
pandoc 论文.md -o 论文.docx --reference-doc=学校模板.docx

# 带目录转换
pandoc 论文.md -o 论文.docx --toc --reference-doc=学校模板.docx
```

| 章节 | 建议字数 | 备注 |

| 中文摘要 | ≥300字 | 必须满足 |
| 第1章 绪论 | 3000-4000字 | |

| 第3章 需求分析与总体设计 | 4000-5000字 | |
